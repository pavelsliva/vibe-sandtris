# AI Replacement Quiz

Мини-викторина на чистом HTML, CSS и JavaScript:

- 15 вопросов о характере работы
- итоговая оценка по трем шкалам:
  - риск автоматизации
  - шанс усиления ИИ
  - человеческий зазор
- результат в формате архетипа, а не грубого `заменят / не заменят`

## Файлы

- `index.html` - структура страницы, карточки викторины и блок результата
- `style.css` - адаптивный интерфейс и визуальный стиль
- `game.js` - вопросы, scoring и логика показа результата
