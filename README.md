# Sandtris Flow

Медитативный антистресс-тетрис на чистом HTML, CSS и JavaScript.

## Что внутри

- `index.html` - адаптивный интерфейс, приветственный экран, элементы управления и мобильные кнопки.
- `style.css` - мягкий lo-fi визуал, responsive layout и оформление панели управления.
- `game.js` - логика фигур, превращение в песок, утекание песка, биткоины, музыка и дыхательные подсказки.

## Как запустить

Открой `index.html` в браузере или подними простой статический сервер:

```bash
python3 -m http.server 4173
```

Затем открой `http://localhost:4173`.
